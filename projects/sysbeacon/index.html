System Beacon

System Beacon, or sysbeacon in short, is a platform operating in a distributed structure, aiming to monitor servers and computers subject to strict regulations, where vital information is processed, without any log loss. System Beacon designed for monitoring entities without data loss on network corruptions. It allows lossless monitoring of all selectable metrics (user activity, service/cronjob logs), especially system performance metrics. Also, sysbeacon is open source. It can be integrated with different platforms. Although it is the recommended installation distribution, it can also work as a monolith.

System Design & Architecture

System beacon is designed in a completely cloud native structure. It can be run in a monolithic or distributed manner according to the need. Agent/metric-miner provides multi-language support for a wide range of infrastructures, from IOT to baremetal servers. Our solution, which has an event-driven architecture, provides easy use with cli, api, grafana and influxDB panel.
